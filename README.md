# Aplypro

Allocation pour les lycéens professionnels.

## Introduction

Cette application servira de support aux personnels de direction pour
renseigner et faciliter le paiement des périodes de stage en milieu
professionnel (PFMP).

## Pile technique

- Ruby 3 ;
- Ruby on Rails 7 ;
- Keycloak ;
- RSpec ;
- Cucumber.

## Démarrage

```sh
docker-compose up
```
